## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/company-registration-portal.git
```

### Install Dependencies

Frontend:

```bash
cd frontend
npm install
npm run dev
```

Backend:

```bash
cd backend
npm install
npm start
```

## 🔐 Environment Variables

Create a `.env` file in the backend directory and add:

```env
PORT=5000
DATABASE_URL=your_postgresql_database_url
```
